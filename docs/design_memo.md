# デザイン・スタイルガイド：design_memo.md

## 1. ビジュアル・ハイライト
- **ファーストビュー:**
  - 「30周年」の祝辞と「鈴蘭のロゴ」を組み合わせ、長年の信頼感を演出する。
  - [cite_start]経営理念「保険を通じて、お客様と社員の幸せを守ること」を、キャッチコピーとして温かみのある書体で大きく配置。 
- **カラーパレット:**
  - 柔らかい黄色（幸福・金運）とピンク（優しさ・スズランの花の色）を基調にする。

## 2. カレンダー・セクションのデザイン
- **更新用管理:**
  - カレンダーは Google カレンダーの公開 iCal から取得して表示。予定の更新は Google カレンダー側で行う。
- **表示:**
  - サイトのテーマカラー（#c2185b / #e8d46a / #faf5e8）で月表示・予定を直接描画。優しい色使いで定休日が直感的にわかる。

## 3. 下部メニュー（フッター）
- **重要文書への導線:**
  - [cite_start]3つの方針ファイル（プライバシー・勧誘・FD宣言）を、フッターに「重要事項・ポリシー」として整理してリンクを貼る。
