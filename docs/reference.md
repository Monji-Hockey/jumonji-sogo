# 参考サイト活用と独自性の統合：reference.md

## 1. ベースとする構造（hoken-aba.comより）
- **信頼感のあるレイアウト:** 保険代理店として必須の「勧誘方針」「推奨方針」「権限明示」等のリンクをフッターに集約する構造を模倣。
- **緊急連絡先の配置:** 事故対応ダイヤルをスマホで押しやすい位置（ヘッダーまたは固定ボタン）に配置する設計を参考にする。

## 2. 独自に強化・変更するポイント（★最重要）
- **30周年のアピール:** - 参考サイトにはない「30周年の信頼」を、トップページの最上部（ヒーローエリア）に大きく配置する。
- **理念の言語化:** - 「保険を通じて、お客様と社員の幸せを守ること」という理念を、単なるテキストではなくサイトのデザインコンセプト（鈴蘭のモチーフ）と融合させる。
- **会社情報の取捨選択:** - 「HP 会社案内.pdf」に基づき、資本金や事業内容は掲載せず、地域に密着した拠点案内（二戸本社・久慈営業所）と「まごころ対応」の姿勢を強調する。
- **カレンダーの独自実装:**
    - 参考サイト以上に、定休日（隔週水・土の不定休）が分かりやすい2ヶ月分のカレンダーをメインに据える。
    - 実装: Google カレンダーの公開 iCal を API で取得し、サイト上に直接描画（iframe 非使用・Cookie 不要）。当月/翌月の切替ボタン付き。
- **分析:**
    - Vercel Analytics（Web Vitals）と、任意設定の GA4（ページビュー・カスタムイベント）を本番のみで利用可能。

## 3. 重要文書の扱い
- 添付された「プライバシーポリシー」「勧誘方針」「業務運営方針（FD宣言）」の3つのドキュメントは、参考サイトの同等ページをモデルにしつつ、提供された最新テキストを正確に反映した独立ページとして作成する。
